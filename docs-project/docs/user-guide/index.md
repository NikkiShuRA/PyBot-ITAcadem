# Руководство пользователя

Этот раздел собран для двух задач:

- быстро запустить проект локально;
- понять основные пользовательские сценарии бота.

## Разделы

- [Быстрый старт](getting-started.md)
- [Установка](installation.md)
- [Конфигурация](configuration.md)
- [Примеры использования](examples.md)

## С чего начать

1. Установите проект по инструкции из [Установки](installation.md).
2. Заполните `.env` по [Конфигурации](configuration.md).
3. Запустите бота локально.
4. Проверьте сценарии `/start` и `/profile`.
