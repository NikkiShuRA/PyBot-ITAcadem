# Примеры

## Базовый сценарий пользователя

1. Пользователь пишет `/start`.
2. Бот предлагает пройти регистрацию.
3. После успешной регистрации пользователь открывает `/profile`.
4. Администратор или ментор может начислять баллы и менять роли в рамках разрешенных команд.

## Примеры команд

```text
/start
/profile
/help
/info
/ping
```

## Примеры админ-команд

```text
/academic_points @user 100 "за решение задачи"
/reputation_points @user 50 "за помощь коллегам"
```

## Пример локальной сборки документации

```bash
just docs-install
just docs-build
```
