# Быстрый старт

PyBot ITAcadem помогает участникам клуба зарегистрироваться через Telegram, посмотреть профиль, получать роли и работать с системой баллов.

## Что нужно пользователю

1. Найти бота в Telegram.
2. Открыть личный чат.
3. Выполнить `/start`.
4. Пройти регистрацию по номеру телефона.
5. После регистрации использовать `/profile` для просмотра профиля.

## Что увидит пользователь

- базовую регистрацию и привязку Telegram-аккаунта;
- профиль с текущими баллами и уровнями;
- команды справки и системной информации;
- роль-зависимое поведение для студентов, менторов и администраторов.

## Основные команды

| Команда | Назначение |
| --- | --- |
| `/start` | вход в сценарий регистрации или переход к профилю |
| `/profile` | подробный профиль пользователя |
| `/help` | список доступных команд |
| `/info` | информация о проекте |
| `/ping` | health-check команда |

!!! note "Важно"
    Команды начисления баллов и управления ролями доступны не всем пользователям. Ограничения задаются ролями и middleware.
